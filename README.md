# Fraud Detection using Machine Learning

## Overview
This project aims to detect fraudulent financial transactions using machine learning.

## Dataset
The dataset contains transaction records including amount, transaction type, and account balances.

## Methods
- Exploratory Data Analysis
- Feature Engineering
- Logistic Regression Model
- Model Evaluation

## Technologies
Python, Pandas, Scikit-learn, Streamlit

## Application
A Streamlit application allows users to input transaction data and obtain fraud predictions.
